0.9.10 Patch Notes

Official Announcements
Ellie August 26, 2017, 12:19pm #1

General:

  • Added Butler, Drunk, Hunter, and Physician rooms.
  • Added a separate chat for Link Minds.
  • Players should no longer blink during death recaps.
  • Added a private note system.
  • Trollbox has been removed from the game as people have been abusing it to make it seem like people are breaking the rules. It will return once we have a reporting system in place.

Possessor:

  • Will no longer meet with the Unseen/Cult at night.

Psychic:

  • Link Minds has been changed. You now target two players. All three of you will be able to communicate together. If you target another Psychic or someone they already linked minds with, you won’t link minds with that player.
  • Link Minds no longer silences targets.
  • Now unique.

Bug Fixes:

  • Some.
