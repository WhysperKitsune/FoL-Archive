While the EV project can be useful for some purposes, their calculations explicitly assume that eliminations occur at random among non-mechanically-cleared players, which in a Mountainous game is all of them. This means that it will consistently return the result that any plausible-to-run setup is mafia-sided; after all, a setup that had a town expected winrate of 50 percent if town eliminates at random would be extremely townsided after accounting for the possibility of reads. (The wiki page notes that on MS, town has historically performed worse than rand in Mountainous, but if town is literally voting at worse odds than rolling a die they deserve to lose)

Mountainous is one of the more wolf-sided commonly run setups (as demonstrated by the number of reasonably balanced 15 player setups that are more townsided than Mountainous), but it is less wolf-sided than a pure expected value calculation would suggest.
