Imo
Co-hosts need to have full access to anything host-related
In case the host has no time, they can decide to post instead for him
Co-hosts can post Voting Counters, they can start and end the day, or do the flips
Co-hosts can answer questions, but the host has the last word and can overrule them
The host should always at least double-check the night results, before giving them out, and important decisions should be discussed with the host (if the host is available)

Co-hosts can do almost everything, but they don’t need to. Showing effort is appreciated tho, and it’s a great learning experience
