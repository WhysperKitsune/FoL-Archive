So basically you use the last 1-2 letters of a word to start off the new word. Example:

Pie
Pienvelope
Pienvelopecodisaster
Pienvelopecodisasteretribution
Pienvelopecodisasteretributionline

Etc Etc

Rules:

1: Always follow the Fourm Rules

2: Don’t change a chain just because you don’t like it

3: Always use the LAST letters of a word

4: Try not to use words already in the chain.

5: Try not to use vulgar language

Have fun
