  15. Make a player lose all their tokens (doesn’t include roulette tokens or special tokens). 1 use.

Today’s event

Consolation prize: At the end of the day, players that didn’t get any abilities from the pool will get a random ability.

Today’s roulette

Roulette will end in 24 hours, you won’t be able to buy tickets after that.

Available Tickets:
Each ticket costs 1 token.

Gambling ticket: This ticket allows you to place a bet on the roulette. You can bet for both a color and a number. Depending on which you get right you will get a prize.

  • Guessing the color correctly gives you 2 tokens for each ticket.
  • Guessing the number correctly gives you 3 tokens for each ticket.
  • Guessing both the color and the number correctly gives you 6 tokens for each ticket.

You can buy as many gambling tickets as you want, but you may only place one bet.

Jackpot ticket: Every token used to buy tickets from the roulette will be added to the jackpot (roulette tokens count as normal tokens for this purpose). The winner ticket will get all the money in the jackpot. Jackpot starts with 5 tokens.

You can buy as many Jackpot tickets as you want, but there will only be one winner.

Finally, I’d like to remind everyone of the following mechanics which weren’t in the first poker mafia or were different:

Selling back

If you don’t like one of your abilities you can sell and it will go back into the ability pool. People will be able to bid for said ability like normal, but the tokens from the highest bid will go towards the seller. Sold abilities will have keep the same amount of remaining uses. You can only sell abilities during the day.

To sell an ability send a message to the host telling him which ability you want to sell. If no one buys your ability, it will be lost.

Using multiple abilities

There is no cap to the number of abilities a player can have at the same time, but they can only use one day ability each day and one night ability each night, however every player can pay 3 tokens to use an extra ability. You can do this multiple times but it will cost you 3 tokens each time. You can’t use the same ability multiple times per day/night.
