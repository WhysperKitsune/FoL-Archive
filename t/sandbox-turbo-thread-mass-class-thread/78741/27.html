O24

Candlemaker

Every night you may kill someone.
The candle’s size starts at 6. If a candle reaches past size 12, the candlemaker will start a new candle on size one. If a candle reaches below size 0, the new candle will start on size 12.
You are informed of the candle’s size on each daystart.

If the victim is Yellow Poppy, the candle rises by 1.
If the victim is White Primrose, the candle rises by 2.
If the victim is Orange Lily, the candle melts by 3.
The candlemaker gets an extra life if a candle reaches size 9.
The candlemaker dies if a candle reaches 3.

Bring your candle’s size to exactly 12 or be one of the last two players standing.
