1.0.4 Patch Notes

Official Announcements
Ellie October 13, 2017, 9:54pm #1

General:

  • Player’s will no longer show up in their room during the death recap. They will now show up in places around the castle. For now there are only 8 spots, but more will be added as time goes on. Feel free to suggest ideas.
  • Prince’s prison has been reworked some. This might help with crashes some people were having. If you crash in the prison, try the lowest quality setting and let us know if that fixes it.
  • Added a little in-game tutorial that displays to players in their first 3 games.
  • Added a display in the top right that says the possible number of each faction in the game. It will not display the possible number of each class type or what classes are in the game. This will only be displayed if you have the Extra Info setting on.
  • Added more info the Royal Blood passive. Now clearly states that it is kept when converted to the Cult.

Hunter:

  • Wolf will no longer attack jailed players. Hunter will not know.

Bug Fixes:

  • The healing flash and blank message when you didn’t require any healing has been fixed.
  • Sometimes the MM couldn’t become an Assassin when using Dirty Work, this has been fixed.
  • Fixed the 16th player’s button in the player panel.
  • Scorned no longer wins every game.
  • A bunch more.
