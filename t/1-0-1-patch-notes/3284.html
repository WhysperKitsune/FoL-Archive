1.0.1 Patch Notes

Official Announcements
Ellie October 1, 2017, 4:02am #1

General:

  • Improved load speeds.
  • Some other optimizations.
  • Added groups. You can now create a group and then join games or start one from your current group.

Bug Fixes:

  • Fixed random armor bug.
  • Messages for certain classes now correctly appear at the beginning, such as the King’s faction message.
  • Some reconnection bugs fixed.
  • Some others.

Added in a word filter for messages, since people can’t not be racist. For the time being the messages just won’t send. It filters mostly racial slurs and a few extreme insults

Notes for 1.0.1a

  • Added sounds for different messages sent by players.
  • Added sounds for being attacked and being healed at night.
  • Added a sound for when the Psychic links minds with you.
  • Added an option to disable the fps display in the settings.
  • Added shortcuts for the journal, deathnote, and private notes. They can be changed in the launcher.
  • J opens Journal by default.
  • D opens Deathnote.
  • N opens Private Notes.
  • Escape can be used to close most of the UI popups.
  • Improved login screen stability
