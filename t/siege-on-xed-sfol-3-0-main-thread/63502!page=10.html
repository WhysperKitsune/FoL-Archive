Sailor, Buccaneer, Cabin Boy and Swabbie have been released.

The Sailor

Green Kraken Investigative
Sonar System (Day) - Choose what you are going to find using Green Flare/Bright Flare tonight. Killer/Offensive, Support/Social, Investigative/Special or Neutral/Unseen/Cult. By default this ability will start on Killer/Offensive.
Green Flare (Night) - Find out if targeted player fits the criteria of your Sonar System ability. - Infinite uses
Bright Flare (Night) - If targeted player fits the criteria, find out what specific part of it they match. - 2 uses
Expedition (Plunder) - Find out target player’s exact class type and faction.

Converts into The Buccaneer or The Seeker
Cult Points: 2 - Each ability costs: 1

The Buccaneer

Unseen Investigative
Sonar System (Day) - Choose what you are going to find using Purple Flare/Shadelight Flare tonight. Killer/Offensive, Support/Social, Investigative/Special. By default this ability will start on Killer/Offensive.
Purple Flare (Night) - Find out what classtype target player is if they match your Sonar System Criteria. - Infinite uses
Shadelight Flare (Night) - If targeted player fits the criteria, find out their exact class. Additionally, this will frame the target to Investigators. - 3 uses
Expedition (Plunder) - Find out two target player’s exact classes.

Developer’s Notes:

I honestly thought this version of The Sailor would be the best idea for this class. Leading to a more customisable Investigative class. Yes, even though the plunder ability is nerfed from it’s previous form and obviously not as good as 2.0 First Mate.

First Mate will be getting a huge nerf with the plunder ability. Don’t you worry.

Buccaneer was a rather interesting class to remake too, it had to be not complete useless shit so it took me a while to work on it.

The Cabin Boy

Green Kraken Support
Inspired (Passive) - You will show up as your targeted player’s class type to Investigative results.
Interested (Passive) - If the player you targeted with your day ability is dead. You will gain an extra use of Invitation
Invitation (Day) - Select target living player, your night ability will be dependant on what class type they are. You will not be alerted to what they are. This ability will be reset to nobody if targeted player dies. - 2 uses
Apprenticeship (Night) - Target player will be affected by the below. The most feedback you will get is a success. - Infinite uses
Graduation (Plunder) - Become target dead player’s exact Green Kraken class. The only exception to this rule is that you cannot use this on a Unique Class or another Cabin Boy, Swabbie or Ritualist!Cabin Boy.
Your objective is to defeat the Unseen or Cult and any Neutrals that oppose you

Converts into The Swabbie or The Ritualist
Cult Points: 3 - Each ability costs: 1
Graduation (Plunder) - Become target dead player’s exact Cult class. You may also pick the Cult Alts that you get from them. The only exception to this rule is that you cannot use this on a Unique Class or another Cabin Boy or Ritualist!Cabin Boy.

Cabin Boy/Ritualist!Cabin Boy Apprenticeship Affects

Killer - Guard targeted player, killing off any attackers trying to attack them. This ability will show up as a success if you successfully killed off an attacker.
Offensive - Redirect targeted player into you. This ability will show up as a success if you redirected an attacker into you.
Support - Heal target player from wounds, nightshade or bleeding. This ability will show up as a success if you healed somebody.
Social - Tomorrow your vote will count as two for execution or pardon. This ability will never show up as a success.
Investigative - If targeted player is a member of the Unseen or Cult. This ability will show up as a success.
Special - Nothing happens

The Swabbie

Unseen Support
Inspired (Passive) - You will show up as your targeted player’s class type to Investigative results.
Interested (Passive) - If the player you targeted with your day ability is dead. You will gain an extra use of Invitation
Invitation (Day) - Select target player, your night ability will be dependant on what class type they are. You will not be alerted to what they are. This ability will be reset to target nobody if targeted player dies. - 2 uses
Apprenticeship (Night) - Target player will be affected by the below. The most feedback you will get is a success. - Infinite uses
Graduation (Plunder) - Become target dead player’s exact Unseen class. The only exception to this rule is that you cannot use this on Unique classes or a Cabin Boy or Swabbie.

Swabbie Apprenticeship Affects

Killer - Prevent visitors from targeted player. This will show up as a success if you blocked someone away from a visit.
Offensive - Redirect targeted player into you. Additionally you are immune to any attacks you redirect and you will show up as Green Kraken to any Investigative abilities you redirect. This ability will show up as a success if you redirected an attack or Investigative ability into you.
Support - Prevent visitors from targeted player. This will show up as a success if you blocked someone away from a visit.
Social - Tomorrow your vote will count as two for execution or pardon. This ability will never show up as a success.
Investigative - Warp targeted player’s night results. This will show up as a success if you made an Investigative player see somebody as a member of the Unseen.
Special - Nothing happens

Developer’s Notes:

An idea I had for a more in depth Xed version of The Squire from FoL was a class that would not know exactly what they were doing but would be able to figure it out if they were good enough.
I found this an interesting concept so I decided to add it in.
