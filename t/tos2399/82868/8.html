Basic Rules

  • Each day, there will be a number of Incant.json files for sale.
  • You may purchase these files with special IncantationNation.arg abilities. These do not count as your day or night action.
  • There are two basic kinds of IncantationNation.arg action:
    • Purchase - Submit 5 security codes.
    • Download - Submit only 1 security code, but gain Download Notes for the file you attempted to acquire.

Security Codes

  • So, what are these security codes? Security codes are a number from 1 to 100 that are submitted to gain files.
  • At the end of the day, the player who submitted the lowest Security Code that nobody else submitted on a particular file will acquire the file.
  • Confused? Here is an example below, simplified, of course:

Three players are attempting to acquire file exam.PLE:

  • Billy Bloobleplop, a wonderful specimen of a man.
  • Plungleberry Dinklefin, professional cat farmer
  • Arete, a nerd.

Billy submits a Download action with Security Code 1.
Plungleberry submits a Purchase action with Security Codes 1, 5, 6, 9, 69.
Arete submits a Purchase action with Security Codes 2, 3, 4, 5, 6.

That means the following security codes are submitted at the end of the day:

exam.PLE - 1, 1, 2, 3, 4, 5, 5, 6, 6, 9, 69

Since both Billy and Plungleberry submitted number 1, the lowest number nobody else chose is 2, and thus Arete gets the file. Billy, however, submitted a Download action, and thus he gets limited information about the file that Arete acquired through the file’s Download Notes
