I think I’d like to see the Private Notes and Public Journal in a side by side format.

I’ve typed stuff in my Private Notes as well but you can only have one or the other up. It’d be cool to see them side by side and if possible maybe have a button next to all information or each page or note that allows you to share it easily.

Maybe a “Share Entry” button. And then when you type a new note in any of the journals (private and public) have a button that says “New Entry” that then produces another “Share Entry” button for that entry. Of course you would have to have a “Share All Entries” button in that case too. But I don’t know… might be too complicated.

The bottom line is, I think we need to have a way to get information that can prove our innocence in a situation out faster. Eliminate as many steps as possible to do so.

Right now its:
Open
Highlight
Copy
Close (I can’t remember off the top of my head but the journal I think can be up when chat is so this is probably not needed but you get the point).
Paste to Chat.

And there is a limited number of characters that can be typed at a time. So to post another bit of the log you have to repeat the process. I understand why there is a character limit and don’t think that should change. But I would like to see a button or link posted in chat that allows anyone that wants to see the log see it. So the information flow would be:

Open
Press “Share”

A lot faster and also allows for the accused to defend themselves longer if need be rather than using up the time to copy and paste logs.
