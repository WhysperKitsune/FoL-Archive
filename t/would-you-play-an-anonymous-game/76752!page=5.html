Icibalus May 22, 2019, 2:57pm #96

Without meta reading becomes far more interesting because you have to get to grips with the alt account’s mindset rather than merely parroting from experience. If you start trying to guess which EXACT person you’re dealing with then having an incorrect read is going to be far more devastating, so instead what happens is you read what KIND of person the man behind the mask is.

This most notably means that you can get away with interesting plays as wolf; unburdened by an exact meta the wolves in an anonymous game make more ambitious plays due to having no established character that they have to somewhat conform to any more, and instead can create a new persona to try out new kinds of play.

This is just conjecture, I haven’t even tried a proper alt account FM game, but there are clear advantages that can lead to a more exciting game when done right. Don’t knock it until you’ve tried it.

(Obviously, we know why anonymous games won’t happen here; it plays havoc with the automod system, the playerpool is too loiw for there to be much doubt of a few particular players being in the game and in any case anonymous accounts can be created automatically on MU so why bother, but it’s a GOAT format nonetheless.)
