Special Gamemode Idea: Custom Classes

Jazz August 31, 2017, 2:49am #1

Now, I’m not saying code every suggested classes for a gamemode, here’s my idea

You enter a class-creator sort of thing and can enter names and factions and whatever(for neutrals you can pick between the Possessor, Fool and Alchemist wincons)

You can then create class abilities as a sort of “If X then Y” sort of thing, X would be something like “Killed” or “Visits a player” or whatever, and Y would be something like “Kill target” or “Occupy King” or whatever

Players can pick whether to make it convert immune or to create a BR class for it or whatever

Players can save their custom classes and put them in games in this one gamemode if they are the lobby host

Example;
Billy goes to the Class Creator and creates the following

Name: Billy’s Sample Class
Faction: Blue Dragon
Type: Support
Day Ability 1: Test - If accused then die
Night Ability 1: Example - Visit player, if Player Class is equal to Reaper then Feedback = “Oh no, a Reaper”
Night Ability 2: Something - If killed then convert self to class Butler

Note: I know this almost certainly would never be implemented, but it’s just a thought

This certainly would allow for massive replayability (besides the already immense replayability the game will have), but as you said, it needs to be extremely well thought-out.

So I say if the devs are okay with the base concept, then you (and maybe other users?) can start fleshing it out and branching out the conditions and how everything behaves with all the possible conditions.

Just be aware that all possible conditions etc would have to be thought of, because an idea like this can only succeed if it is well done – if done in a sloppy manner it would just end up clunky and not fun.
