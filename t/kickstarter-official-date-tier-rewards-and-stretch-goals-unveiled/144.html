what’s up with the Demand Justice group reward?

The only thing we’d START with is to have a report button which will simply email the devs to manually look into it when we can spare the time (less time available during alpha/beta, more time during release).

This is very different than an actual system that could automate this stuff, be smart and discipline them without developer interaction. Everyone would have a hidden set of Reputation points. The more reputable you are, the more your reports count. Maybe even have a tribunal system inspired by League of Legends. We’d probably take community feedback when ready to implement.

The current way doesn’t guarantee anything will actually happen. I call it Ghetto Report. We should probably add that to clarification section.

If we don’t reach that stretch goal, we’ll have an epic system one day, but would need to gather resources first.
