sulit restates things kat already said

I… definitely think this went better than the first game in terms of what we set out to improve on, and I think it went better overall.

But there’s some things I think went wrong as well.

The main thing I feel like I could have done better is that, I just don’t think I had enough time to give this game the love it deserved. We were pretty rushed to begin it as danganronpa begun, then school started and I play soccer, and I feel like I couldn’t spend as much time on it as it really needed. Shout out to kat for getting really good at hosting go kat yay beans.

Everyone was hyped for the game, I was hyped for the game, kat was hyped for the game, I feel like we were all so hyped that we didn’t take the time we really needed to make sure everything was good. I think this mainly shines through in map design.

@katze definitely should not blame themselves for these issues ):<

However, I think we improved a lot and learned a lot about some do’s and don’t’s both map-wise and role-wise that I think will carry over well into the next game. Learning process !!!

As kat pointed out, inacivity was definitely a problem, as a mentioned in the spec chat a lot I feel like the game would have gone a lot different if different players were in different slots. I think introducing some sort of requirement or warning about activity would be a good place to start in the next round, or not being wimps about replacing people

There’s some weird shit people did that I don’t really want to get into because lolsalt, but I definitely do not think fake claims are needed. All alignments have a use for a fake claim at some point in my opinion. If we gave them to evils, we might have to give one or two to neutrals, oh these guys need them too, and then we’re creating a whole other role list thats just, fake claims. It’s wasted energy and it makes the game much more exciting when people have to come up with them themselves >:D

One last thing. I feel like the ability to move unlimitedly had drawbacks, not because we couldn’t process everything, but because the whole map was quickly explored, leaving about nothing for the players that weren’t online when it was opened. So I’d also suggest limiting the amount of movement people can do, or something.

ayaya!
