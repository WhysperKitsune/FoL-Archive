The Animal Lover

Neutral Support

Passive (Innocence) - Immune to Death at Night. Immune to Most Occupation and Redirection.

Night Ability (Fluffy Companion) - Give a player a Cat. Their abilities will have additional effects on the player that has a Dog.(Infinite Uses - Only 1 player can have a Cat - Can NOT Target the person that has a Dog)
Night Ability (Dog Companion) - Give a player a Dog. Their abilities will have additional effects on the player that has a Cat.(Infinite Uses - Only 1 player can have a Dog - Can NOT Target the person that has a Cat)
Goal: Both players that have animals survive until the end.

Effects

Cat Owner Effects on Dog Owner :

  • The Butler can no longer occupy the target.
  • The Court Wizard’s Clear Mind and Empower will have no effect on the target. Tornado can NOT be used on the target.
  • The Drunk can no longer redirect/occupy the target.
  • The Hunter can no longer kill the target with Bear Companion. The target will be immune to Wolf Companion.
  • The Knight can no longer Defend or Coldsteel the target.
  • The Mystic can no longer use Link Minds or Conduit on the target.
  • The Noble can no longer use Gossip on the target.
  • The Observer can no longer follow the target or Window Peek the target.
  • The Paladin can no longer check the target or use Smite on the target.
  • The Physician can no longer heal the target.
  • The Prince can no longer Execute the target.
  • The Princess can no longer Flirt or use Will-o-Wisp on the target.
  • The Sheriff can no longer investigative or use Surveillance on the target.
  • The Alcoholic can no longer redirect or occupy the target.
  • The Aristocrat can no longer use Intrigue on the target.
  • The Assassin can no longer attack the target.
  • The Duchess can no longer investigate the target. If the target is Unseen , Dark Wisp will have no effects on the target.
  • The Enforcer can no longer use Frenzy on the target.
  • The Herbalist can no longer drug or defile the target.
  • The Illusionist can no longer use Mindwarp/Mental Blur on the target.
  • The Marshal can no longer use Probe or Snoop on the target.
  • The Mastermind can no longer convert or use Foresight on the target.
  • The Nightwatch can no longer follow the target or use Watch Carefully on the target.
  • The Poacher can no longer visit the target. Spike Traps will have no effect on the target.
  • The Sage can no longer use Wall of Fire on the target. They can not use Tornado on the target.
  • The Servant can no longer occupy the target.
  • The Apostle can no longer use Mindwarp on the target.
  • The Cult Leader can no longer use Eradicate on the target. They can not bleed the target. The target can not be sacrificed. The target can not be converted by you.
  • The Invoker can no longer occupy/redirect the target.
  • The Ritualist can no longer use Teleportation on the target. If the target is Cult , Brotherhood will have no effect on them.
  • The Seeker can no longer check the target.
  • The Alchemist can no longer visit you.
  • The Fool will not be affected.
  • The Inquisitor can no longer investigate/attack you.
  • The Mercenary can no longer visit you.
  • The Possessor can no longer attack you.
  • The Pretender can no longer investigate/attack you.
  • The Reaper can no longer visit you.
  • The Scorned can no longer visit you or Trollbox you.
  • The Sellsword can no longer visit you.
  • The Sorcerer can no longer attack you.

Dog Owner Effects on Cat Owner :

  • The Butler’s Serve Wine bypasses the target’s Occupation Immunity.
  • The Court Wizard’s Empower will prevent all visits against the target.
  • The Drunk’s Debauchery bypasses the target’s Redirection Immunity.
  • The Hunter can instantly kill the target when using Wolf Companion. If the target attacks The Hunter when he uses Bear Companion , they can not be healed.
  • The Knight can bypass the target’s Night Immunity when using Coldsteel.
  • The Mystic can prevent the target from being converted if they used Link Minds on the target.
  • The Noble will see everything that happens to the target when using Gossip.
  • The Observer will see the target’s class when using Window Peek.
  • The Paladin will always get accurate results when checking the target.
  • The Physician will make the target Immune to Nightshade/Bleeding if they successfully healed the target.
  • The Prince can execute the target without losing a charge. (This applies even if the target is a member The Blue Dragon. You will still have 1 Execute left.)
  • The Princess will always get accurate results when checking the target.
  • The Sheriff will always get accurate results when checking the target.
  • The Alcoholic’s Debauchery bypasses the target’s Redirection Immunity.
  • The Aristocrat will see everything that happens to the target when using Intrigue.
  • The Assassin will bypass the target’s Death Immunity.
  • The Duchess will get the target’s exact class. If the target is a member of The Unseen , Dark Wisp will make them appear not suspicious for 2 nights.
  • The Enforcer can no longer be occupied when targeting the target.
  • The Herbalist will occupy the target after the night they have drugged the target. Defiling the target deletes his logs.
  • The Illusionist will frame the target when using Mindwarp.
  • The Marshal will get the target’s exact class.
  • The Mastermind will always see the class type of the target when using Foresight.
  • The Nightwatch will not use a charge of Watch Carefully when using it on the target.
  • The Poacher’s Spike Traps and Snare Traps will bypass the target’s Occupation Immunity.
  • The Sage will occupy the target when using Tornado.
  • The Servant’s Serve Wine bypasses the target’s Occupation Immunity.
  • The Apostle will frame the target when using Mind Warp.
  • The Cult Leader’s Eradicate will bypass the target’s Death Immunity. If the target is bled , he can not be healed. If the target is sacrificed , The Cult Leader will get 2 uses of Eradicate.
  • The Invoker will bypass the target’s Occupation/Redirection Immunity.
  • The Ritualist will make the target Occupation/Redirection Immune when using Teleportation. The target will be Death Immune when you use Brotherhood on them.
  • The Seeker will get the target’s exact class when checking them.
  • The Alchemist will bypass the target’s Death/Redirection/Occupation Immunity.
  • The Fool will not be affected.
  • The Inquisitor will bypass Deth Immunity.
  • The Mercenary will gain 1 extra Brilder when using abilities on the target successfully.
  • The Possessor will bypass Healing/Death Immunity.
  • The Pretender will bypass Healing/Death Immunity.
  • The Reaper will bypass Death Immunity.
  • The Scorned will not be affected.
  • The Sellsword will bypass the targets Occupation Immunity.
  • The Sorcerer will bypass Healing/Death Immunity.
