Stuck at main menu

Bug Reports
phillies27 October 31, 2017, 5:54pm #1

What happened?

I just bought the game yesterday, and after looking at the tutorial, the rules, the guide, and the classes, I went to find a game and clicked start solo. I then got the message “Timed Out (Reconnecting…)”. It never reconnected. I then tried creating a group, but got the message “Verifying Version…”. It never changed. I then tried to enter an invite code, but got the message “Something’s Wrong: Tell a Dev in Discord”. I verified integrity of game files through steam and it said “1 file failed to validate and will be required”. Tried finding a match, same thing happens. Re-verify the game integrity, same message again. I tried re-verifying the game integrity once more before trying again, and this time it said “All files successfully validated”.

What was SUPPOSED to happen?

The game should have let me found a match solo, create a group, and join a game through the enter an invite code.

Steps to reproduce:

I open the game
Login with Steam
See that the find match box has ? players online, ? rooms(matches), and ? in Lobby.
Click start solo (it is flashing immediately after login with steam) or find game, the Ready text sometimes stays the same or changes to timed out.
Try clicking create group, it returns verifying version…
If I find a code through the discord, I enter it in the enter invite code, and I get an error message.
Exit game and verify integrity of the game files twice.
The first try it repairs a file, the second try it says everything is validated.
Repeat steps.

Comments:

I’ve tried restarting my computer and turning off and on my wifi.
I am on v1.0.7b at the time of writing this, and prior to running all these steps again today, I received a new update to this game (the game wasn’t working prior to the update today). I can post images if needed.
