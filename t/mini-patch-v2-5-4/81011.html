Mini Patch v2.5.4

Official Announcements
í42-Xblade August 10, 2020, 6:13am #1

Mini patch with more bug fixes

Class Fixes

  • Tornado abil can no longer target King in 2nd target slot.
  • Cult Leader’s day ability enable<>disable weirdness was fixed.
  • Fool will no longer win while dead without finishing their objective.
  • Heal abil feedback for Did Not Require and Could Not Heal was inadvertently swapped and is now fixed.

Suicide Abil

  • Scorned may no longer Disguise a suiciding player (they will receive the “already dead” feedback).
  • Toggling Suicide abil will now inform evil allies.
  • Suicide is now shown on Night 2 (from Night 3).

General

  • Swapped lighting to CPU-based instead of GPU-based to prevent FPS drops (GPU was Unity’s experimental feat, anyway). Let me know if improved.
  • Added a {me} variable to custom logbook templater in settings.

Test Realm

  • Name selection timer is reduced to 10s (from ~30s) since names are now auto-selected in PTR.

Dev Notes

This mini patch was a follow-up to v2.5.3 with more class bug fixes. Next patch will likely also be focusing on class bug fixes.

v2.5.4b

  • The new Suicide abil toggle “evil allies” feedback will no longer announce it… to everyone D:
  • Players that change classes (Dirty Work, Convert, etc) should no longer see their abilities when jailed the same night. This isn’t tested, but should do the trick. This was like a 2-year old bug
  • Fixed an awkward Psycho King description at the top.

v2.5.4a

  • April Fools spam @ end fixed.
  • Added missing Inquisitor April Fools portrait.
