I made some changes to poker mafia classes for future poker mafia games.
Differences from the first poker mafia classes:

  • All classes are able to get abilities from the pool.
  • Townies and mafiosos will be able to choose a gift at the start of the game (see below for the gifts they can choose from).
  • Most gambling events have been removed and replaced with the ‚ÄúRoulette‚Äù.
  • There are still daily events, but they no longer involve gambling (see the events section for more information).
  • All classes can use more than one day or night abilities, but they must pay 3 tokens for every extra ability besides the first one. (You can‚Äôt use the same ability multiple times).
  • Godfather no longer needs to use tokens to kill.
  • Godfather no longer buys abilities for his mafiosos, he buys them for himself.
  • There are 8 different ability types: Support, Offensive, Social, Investigative, Killing, Gambling, Passive and Scum. There are also certain classes with starting abilities. These abilities will appear as ‚ÄúSpecial‚Äù to investigators.

Available gifts:

  • Abilty gift: Start the game with a random, non-golden ability.
  • Token gift: Start the game with 3 extra tokens.
  • Golden gift: Start the game with a Golden ticket. At the start of the game, one person with a golden ticket will be selected at random and get a free Golden ability. (Golden abilities are powerful one-shot abilities).

The Roulette
Gambling related events have been replaced with the roulette. The roulette is available all days and offers you the chance to buy tickets for it. There is no limit on the amount of tickets you can buy every day. The roulette lasts for the first 24 hours of the day. Each ticket costs 1 token.

Available tickets:

Common tickets: These are used to bet on the results of the roulette. All 3 common tickets are available every day.

  • Color ticket: Pick red or black. If you are right you win 2 tokens.
  • Number ticket: Pick a number between 1-3. If you are right you win 3 tokens.
  • Full ticket: Pick a number between 1-3 and a color. If both are right you win 6 tokens.

Rare tickets: Out of everyone that buys these tickets, only 1 of them will get a price. Only 1 of these will be available each day.

  • Jackpot ticket: Every token used to buy tickets from the roulette will be added to the jackpot (roulette tokens count as normal tokens for this purpose). The winner ticket will get all the money in the jackpot.
  • Golden ticket: The winner ticket will get a golden ability.
  • Special ticket: The winner ticket will get a special token.

Selling abilities:
The way selling an ability works has been changed. You can now choose an ability to sell and it will go back into the ability pool. People will be able to bid for said ability like normal, but the tokens from the highest bid will go towards the seller.
