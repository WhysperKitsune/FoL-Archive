(Class Suggestion) Morpher

Class/Lore/Fanfic
Htm April 11, 2018, 3:21am #1

Morpher, Neutral, Support

Passive 1: Additional Power - If the class you remember has a limited use ability you will have 1 extra use.

Night Ability 1: Morph - morph into a dead class. If the class is not a MM/CL/Assassin/Scorned/Merc/Inq/Sorc/King/Prince You will remember it and stay as that class for the rest of the game unless you are converted/promoted/chosen as king, does not inform players, can’t be used on the first 2 nights. Unlimited

Night Ability 2: Barrier Morph - Morph yourself as a shield making you immune to attacks for the night. 1 use

Win Condition ~ Remember a class and complete his win con

Additional Power:

~ Gets 1 extra Emerald Potion if morphed Alch
~ Gets 1 extra HH if morphed Alcoholic/Drunk
~ Gets 1 extra Snoop if morphed Aristocrat/Noble
~ Gets 1 extra Concentrated Wine if morphed Butler/Servant
~ Gets 1 extra Tornado if morphed CW
~ All Cult get 1 extra use on their second ability if morphed
~ Gets 1 extra Dark Wisp if morphed Duchess
~ Gets 1 extra Frenzy if morphed Enforcer
~ Gets 1 extra Hide if morphed Fool
~ Gets 1 extra Defile if morphed Herb
~ Gets 1 extra Wolf if morphed Hunter
~ Gets 1 extra Mental Blur if morphed Illusionist
~ Gets 1 extra Expose if morphed Marshal
~ Gets 1 extra Conduit if morphed Mystic
~ Gets 1 extra Window Peek/ Watch Carefully if morphed Obs/NW
~ Gets 1 extra Smite if morphed Pally
~ Gets 1 extra Self Care if morphed Phys
~ Gets 1 extra Snare Trap if morphed Poacher
~ Gets 1 extra Possess if morphed Possessor
~ Gets 1 extra Wow if morphed Princess
~ Gets 1 extra Soul if morphed Reaper
~ Gets 1 extra Wall of Fire if morphed Sage

I know this is like ToS’s amne but I think a concept like that would be really interesting and make very interesting new strats for evils and also give the Poss/Reaper a bit of hope that they might win if Morpher morphs into them

Feedback/suggestions are appreciated!
