Dooku August 31, 2017, 2:48am #1

If you could somehow implement one of the numerous translation APIs out there, it would surely be useful. If it were implemented, you wouldn’t have to worry about region specific servers which separate the different player bases. I.e. With a translation API implemented into the game’s linguistic processor, the doors open for more connection of peers- it would automatically translate the languages into the player’s native language, and vice versa. I.e. A (for example) Russian could talk to someone from Portugal without even knowing, which could possibly also prevent prejudice.

If you google “Translation API”, you will see that many APIs designed for just this purpose have already been developed.

I believe it would be well appreciated.
