Shrimpy November 13, 2018, 8:45pm #1

Some of these ideas are not necessarily my own but anyways here goes. These are things I deem to be major problems as opposed to simply things which could be improved but don’t need to be:

  • Allies OP

I think almost everyone agrees on this point. It hurts neutrals so badly and also pressures them to open-claim neutral which goes against the spirit of the game.

  • Observer OP

If Follow were reverted into how it used to be (as I hear it was split into two weaker abilities), the problem would disappear.

  • Chronomancer OP

Chrono is very close to being a killer Physician who can self-care and heal in the same night.

  • Posswords

Giving Possessor the whisper history of his Possess target would eliminate this tedious meta and is a better solution than the current whisper-reading one for Puppet-Strings. Also, give him the entire chat and ability feedback history of his target.

  • Prince Exe & Coldsteel unusable before n3

The only reason this exists is because of gamethrowers, but your anti-gamethrower system supposedly works very well so get rid of this. It encourages neutrals to open-claim D1 as well as letting Fools (and people pretending to be Fools, like Sellswords) to run amuck for 2 days.

EDIT:

  • Noble twin-vote confirmation

Noble twin-vote notification should not be public if it doesn’t affect the trial’s outcome. According to my philosophy, any ability whose primary use (in practice) is confirmation, needs to be changed to eliminate this practice. The current primary use of twin-vote is confirmation, so it needs to be fixed.
