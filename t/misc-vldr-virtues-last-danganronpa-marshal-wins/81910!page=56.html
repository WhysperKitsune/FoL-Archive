sulit June 15, 2020, 5:12pm #1106

Alrighty so, gonna point out some things that game that I coulda done better/will be fixed for next game.

First off would like to say that this game was so much fun to host and design even if it was stressful sometimes. I very much enjoyed hosting it and would love to again after making changes. I hope y’all had fun playing it :D.

Roles

First thing I’d like to point out are the roles. I took em all from the ToS page this game as I wanted to test the waters and see how it’d work out. While some roles worked pretty well, other roles seemed pretty boring or hard to play compared to others. Next game I plan to just make custom roles and make more interesting and diverse wincons for each one, including neutrals. I think this will make the game more fun to play for noots and add a new sort of dynamic to it. There’s definitely a lot of area to grow roles-wise.

Instakills/knocks

Uh, kinda self explanatory. Really unfun for the player getting knocked, was an idiot move on my part to add mechanics that knock people with such ease. Will not be in the game next time, I’m going to try and work out the kinks combat-wise.

Role/Wincon Claiming/Speculation

Definitely removing this rule and will instead give necessary fake claims to the roles that need them, as Alice suggested. The modkills really hurt to do this game and I regret doing every single one. I don’t want to do that again and I think that enforcing such a rule as this, on this forum, just is not going to work out.

Those were the big things I wanted to address. I think this game went well but was more of a test run in terms of running it on these forums. I think it really helped work out some issues the game had and I think it will run much better next game.

If anyone else has any ideas or anything they think would be cool to add or remove before I start working on designing the next one I’m all ears.

Kat and I were also discussion hosting Roaming via discord and I want to know what y’all think about that.
