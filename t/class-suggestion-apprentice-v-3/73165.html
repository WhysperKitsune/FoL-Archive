I know i had this class before but the meta has changed a lot, so i want to see what you think of this class being played in the game now

     •Apprentice

Type - Support
Faction - Neutral

• unique

my favorite classes to come up with are Neutral. Mostly because they offer a larger veritity of option and some very unique win cons. This class has a simple win con but adds some depth to it.

CANT SPAWN IN WITH SS

Passive 1:
Known allies - if any evil doer (NK/Cult/Unseen) visits you then you will know who they are and what class they are. You will also become death immune that night

Passive 2:
Transformation - if you select a class with your Mimic day ability and that class has redirection or occupation immunity, then you will gain it too for the night

Day ability 1:
Mimic - pick a player, you can use their primary ability tonight (the ability on the left) (unlimited uses) if the player is a unique or special class then you will be notified and will not be able to use their ability, instead you will become death immune that night . Can not target the same player more then 3 times

Day ability 2:
Hidden powers - (2 uses) - Pick a player, and disguise them as any class of your choice

Night ability 1:
New toys - the ability that you mimicked this morning will be the ability that takes this night abilities place. @Kirefitten

Win con - Defeat the BD at any cost

Questions that you could answer for this class:

1 - will this be a good class to play along with in the game?
2 - will you have fun playing this class?
3 - do you think this class is too overpowered or too underpowered?
4 - do you think this class will change the game in a positive or negative way?
