More changes are inbound, but these are some to start us off:

Globally

After being redirected/occupied 3 times, you are redirect/occupy immune.

After being jailed 3 times, you are jail immune.

The Physician

New Exhume (day) - At the end of the night, learn what faction killed a player. - 2 uses

Cult Alt
(Exhume): Exsanguination (day): At the end of the night, learn what classes killed a player, or if they commited suicide. - 2 uses

The Maid

Nosy Servants removed.
Added new day ability Change Sheets (day) - Target two corpses. At the end of the night, learn if they were killed by the same player. - 2 uses.

Cult Alt
(Change Sheets): Mithras’ Wisdom (day): At the end of the night, learn if a flip has been tampered with and by which class. - 2 uses

The Herbalist

Exhume changed to

Disembalm (day): At the end of the night, learn what classes killed a player, or if they commited suicide. - 2 uses

The Handmaiden

Added new day ability Search Room (day) - At the end of the night, learn if a flip has been tampered with and by which class. - 2 uses.

The Assassin

Hidden Blade number of uses reduced to 3.

The Cult Leader

Rupture number of uses reduced to 3.

The Fool

Trickery now silently redirects everyone who voted to execute the fool to themselves.

The Fanatic

Changed name back to the The Sellsword
