Because replacing someone unable to play with someone able to play may be helpful

But look

A: We absolutely shouldn’t make “can I barely meet the minimum posting requirements” our target for playing games

B: The idea behind the game is that each slot is active and competitive — we force-replace slankers because they actively detract from the game

C: Ultimately it’s much better if you can tell me “yo I’ma need a replacement please start looking” then if you force me to wait until you’ve spent multiple days not posting to find one

D: While replacing in a stronger player can somewhat swing the game, that’s going to be much more fun for everyone than a dead game would have been

E: There are already measures in place to punish people who try to abuse this privilege

All told I really don’t think the system needs any change. If you think someone is abusing replacement, report them, if you think players are are too inactive and need too many replacements, go play on another site.
