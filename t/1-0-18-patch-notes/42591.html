1.0.18 Patch Notes

Official Announcements
Ellie February 5, 2018, 2:12pm #1

Butler:

  • Nightshade Wine will now kill the King the same day he is poisoned, unless healed. Can now be used before Day 4.

King:

  • Can now be voted up before Day 4.
  • If the starting Good King is executed, then the court will be unable to accuse anyone of treason for a day.
  • Order Execution - Now only bypasses death immunity, and can be used before Day 4.

Mastermind & Cult Leader:

  • After failing to convert someone two times in a row, if the next conversion fails, it will instead target a random convertable player.

Sheriff:

  • No longer unique or unconvertable.
  • Surveillance has been removed.

Physician:

  • Death Knights will now know who reanimated them.
