FoL balance changes

These changes will go into effect prior to FoL 28

All Blue Dragon classes have been re-worked into the Peasant, which is allowed to speak and vote

Player size has been reduced to 15

Cult has been removed

Conversion has been removed; to compensate, the Unseen start with a third starting member, the Angry Peasant.

The Mastermind may still mimic any Blue Dragon class.

The Assassin’s Assassinate, 2-for-1, Lacerate, and Distract abilities have been reworked into an ability that allows the Unseen to, each night, Douse a target, and 3 times per game ignite all doused targets. Additionally, the Assassin gains a passive that allows it to use both abilities simultaneously.

If the Assassin dies, the Angry Peasant will become the new Assassin, retaining any Douses. If the Assassin and the Angry Peasant both die, the Mastermind will likewise become the Assassin, retaining any douses.
