Mathematician - BD invest
Passive: Focused - Immune to ability’s that change target.

Night ability 1: Calculate - Check if the first targets ability would work on the second (unlimited)
Night ability 2: Inspect - Check if the target had an ability used on them that failed (unlimited)

Researcher - Unseen Invest
Passive: Focused - Immune to ability’s that change target.

Night ability 1: Study - Check what message the second target would receive from the first visiting him. You may target unseen members if you want (unlimited)
Night ability 2: Inspect - Check if the target had an ability used on them that failed (unlimited)
