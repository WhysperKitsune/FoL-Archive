Yes, @Luxx and @newarkinvaders this was drawn with a program that I coded, I was just posting it here because I figured “why not”. However, there is nothing to fear in terms of harm to the community regarding deathnote’s, I will NOT be releasing this program to the public. EVER. The sole reason I created this was for fun, and personal use. Nothing more than that. I just like the reactions people provide in games that I play from the deathnotes my code creates. Theres nothing more to it.

The main reason I will NEVER release my code for the program is solely because it is more than capable of redrawing images with color in a black and white format using shade thresholds. Oh btw in reponse to the lines not being clean: this is due to the different drawing methods i implemented into the code. I had this picture drawn using a rotational function, so the lines were drawn by the mouse moving in small little circles along the line.
