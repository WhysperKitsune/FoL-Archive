I’ve done this, in my case I killed 2 people in one night, which is probably what happened for you.

In my case I killed the Assassin and Possessor attempting to jump in the same night.

The messages are independent kills.

“You killed someone while guarding X last night.” = kill that you survived because of Armored
“You died guarding X last night.” = kill that you didn’t survive.

Normally you don’t see both messages at the same time. Normally you’d only see one of them.
