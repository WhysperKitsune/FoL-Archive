The Builder (class suggest) Revamped

Class/Lore/Fanfic
NOBLE1 August 9, 2018, 6:43pm #1 
              •Builder 

     **_Work in progress... still_**

Description: this is a class built heavily around different constructions and when to make them. Each construction has different attributes and is dangerous to visit.

Mechanics - only one builder per game, the builder will only spawn in 16 man games. If the builder dies with a construction still up then the building will still remain.

All constructions can be built in anyone’s room. Everyone can visit that player as normal but depending on the building something else could happen

Type - special
Faction - Neutral

•The Builder will know what faction he is working for.

Passive 1:
Hard worker - can use both day abilities on the same day

Passive 2:
Long hours - When you are attacked your first time, you will be immune because you spent extra hours at work. If you are brought to jail then the construction will be stalled for a day.

Day Ability 1:
Construction - select a building to start working on, it will be finished the night you select it. Then select a player to build the room in, it will stay until destroyed or a new building is made. (Unlimited uses)(1 day cool down)

Church -
If any BD member visits the room where the church is then they will be healed of any poison/bleeding.

If any Unseen visit then everyone who visited the room that night will bleed

If any Cult visits then everyone who visits the room that night will bleed

Blacksmith - anyone who visits the blacksmith will regain all uses of their limited abilities

Stable - if the builder selects the stables at night then the time will be increased for an extra 10 secs during the day and is able to have an extra trail during the next day.

Dungeon - If king sends a guard to the dungeon then the prince will be night immune.

If any Unseen/Cult visit the dungeon then the prince can’t exe his target that night.

Library - If anyone visits the player with the library then they will gain a passive based off of the room where the library is. You will not know you gained any passives

  • if the library is in a room where the player has an occu immune passive then anyone who visits will be occu immune for the next night.
  • if the library is in a room where the player has a death immune passive then they will be death immune the next night
  • if the library is in a room where the player has both death and occ immune passive then they will gain both the next night.
  • this can work with hunters passive and any other passive I did not mention.
  • (Can only be built day 3 and beyond)

Torture chamber - if anyone visits the chamber then the next night they will be occupied, do to the disturbing thoughts they saw the night before.

_reinforced doors_🚪 - anyone who visits this player will be prevented. This will stay for two night and then will be destroyed.(1 time use for this construction)

Witch craft room - if any BD visits this room they will be empowered(normal Cw empowered would effect the person)

If any Unseen or Cult visit the room then they will appear as non sus to invest results.

If any Neutrals visits the room then they will be death immune that night.

Day Ability 2:
Private matters - same as mystic (unlimited uses)

Night ability 1:
Secret door - select yourself, anyone who visits you will be redirected to where you built your building if you have one. If you don’t then you may not use this ability.(2 uses) {you may select the stable if you have it built with this ability}

Night ability 2:
Loose nails - undo the nails collapsing the building on any players who decided to visit the current building, making them all bleed. The current construction will be destroyed. (Unlimited uses)

Win condition - have the starting king and his faction lose to win.

Lore: The King decided to cut your pay for his own benefits. Now the builder is out for revenge.
