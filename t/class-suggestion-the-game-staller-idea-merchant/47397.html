This idea was not started by me. It was started by @Tormental I believe and continued/reworked by others like @Livicus and @NightX (I’m only hopping on the bandwagon lul)

Name: Merchant(dunno could be renamed into something different) Neutral Offensive

Passive 1: Clever - Occupation and redirect immune.

Passive 2(still thinking about this, might remove it): [Name Needed] - If you die by prince or king execution then there can be no voting tomorrow
Feedback/Reasoning: It seems slightly useless but could help if you are found early. Remember, you are like a mercenary, so you don’t have to live in order to win.

Day Ability: Bribe(bad name?) - Today’s trial will be force pardoned/executed(depends on what if choose) and there can be no more trials when this is used if you chose to pardon them and this is the first trial. Does not work if you are out up. Can be Vetoed by the King’s Decide Fate. [1 Use]
Feedback/Reasoning: It’s to cause more chaos and put some doubt onto the king ("why’d you exe/pardon them?) And if it’s BD king, they won’t be allowed to vote for 1 day. I can change to force pardon only if it seems too unbalanced.

Day Ability 2: Trollbox - Everyone knows what this is.
Feedback/Reasoning: It’s so you can make people say things that will hopefully hesitate/be confused for longer

Night Ability 1: Haggle - Redirect someone to you(bypasses occupation immunity if they are a killer class). You will be Immune to all attacks. Can only be used on each target twice. You will know if you prevented them from acting. (once per target if you feel this is too op against the mm/nk) [Infinite Uses]
Feedback/Reasoning: This is so you can kinda reliably claim drunk and protect yourself/stall the game cause pretty much everyone wants to kill this class as getting occupied/redirected is annoying af

Night Ability 2: [Name Needed](Vigor Elixer and Snake Oil is a name @Tormental came up with) - Make a person Death Immune tonight in exchange for Occupying them(bypasses occupation immunity). Can only be used on the same target twice. You shall know if you prevented them from acting or if you protected them [3 Uses]
Feedback/Reasoning: Maybe also occupies any killers visiting you or makes you death immune the night you use it so it can help you live longer. Also this is so evils/investigators can’t convert/find evil which can help stall the game, depending on who cutrently has the upper hand.

Win Condition: Stall the game to D8(if game ends on D8, it still counts)
