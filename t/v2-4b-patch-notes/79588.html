v2.4.0b - Hotfixes

Fixed many - but not all new bugs. Note that test realm was fixed for expediency, so let us know if some of what’s listed isn’t true:

  • Fixed Chinese UI issues where you couldn’t play or create a game.
  • Fixed some game over issues with winners and the winners display.
  • Possible fix of redirection visual bug.
  • Fixed “NoClass” bug for Kings - may ripple to other fixes (it wasn’t just end-game).
  • End-screen and winners should now show specific types of Kings.
  • Pretender and Neut King should now be able to win.
  • Sellsword can now win with evil
  • Sorc is now immune to bleeding.
  • Royal Blood has returned to royal classes, fixing several abilities.
  • Pretender’s Ballot Mixing abil should now work properly.
  • Inquisitor should have 3, unique Heretic targets (and should be able to win - please confirm with us if you win)
  • Aristorcrat typo fix.
  • Alch’s Stoneskin is now usable after trial.
  • Single abilities are now centered.
  • Fixed abnormal “Psycho” classes.
  • Trollbox should now be able to use during trial (Please confirm if you try this).
  • Should have fixed Unique Mystics.
  • Fixed a chain of Hunter Retribution (Revenge) bugs.
  • The new Teemo+Mithras avatar now shows Teemo properly having a white chest
  • Follow can no longer target the King
  • Follow abil now has memo that you can’t target King
  • @ Guides: Fixed report bugs
  • Moved MVP panel over slightly to the right to show it’s done on purpose for those that like a big chat box.
